Plan semestral (materias ofertadas)

Programa ofrecido el semestre actual

/programa-disponible get

get

Recupera datos de las materias ofertadas en el semestre. ES decir las materias con las que se puede hacer el horario.

get /programa-disponible

Recupera datos de las materias ofertadas en el semestre. ES decir las materias con las que se puede hacer el horario.

  • Request
  • Response

Headers

  • access_token: required(string)

    el token devuelto al iniciar sesion (Authorization bearer)

Query Parameters

  • carrera: required(string)

    Acrónimo de la carrera Posibles valores (ICC, LCC,ITI) Pasar solo este parametro retornará solo los nombres de las materias

    Example:

    https://hidden-wave-33419.herokuapp.com/api/programa-disponible?carrera=ITI
  • nivel: (string)

    Tipo del plan. Si se omite se retornan todos los niveles. Valores permitidos (formativo,basico,optativas)

    Example:

    https://hidden-wave-33419.herokuapp.com/api/programa-disponible?=ITI&&nivel=formativo
  • mat_id: (boolean)

    Pasar como true si se requiere el identificador de cada materia.

  • profesor: (boolean)

    Pasar como true si se requieren el profesor que imparte de cada materia.

  • horarios: (boolean)

    Pasar como true si se requieren los horarios de cada materia.

HTTP status code 200

Body

Media type: application/json

Type: object

Properties
  • carrera: required(string)

    nombre de la carrera a la que pertenecen las materias

  • nivel: required(string)

    nivel al que pertenecen laaaas materias. Valores posibles (todos,basico,formativo,optativas)

  • materias: required(array of )

    lista de materias disponibles con las propiedades especificadas en la query

Examples:

default:

{
  "carrera": "ICC",
  "nivel": "todos",
  "materias": [
    {
      "asignatura": "DHPC",
      "nrc": "2653"
    },
    {
      "asignatura": "Matemáticas",
      "nrc": "2654"
    }
  ]
}

?nivel=formativo&&mat_id=true:

{
  "carrera": "ICC",
  "nivel": "formativo",
  "materias": [
    {
      "asignatura": "DHPC",
      "nrc": "2653",
      "mat_id": "FGUS 002"
    },
    {
      "asignatura": "Matemáticas",
      "nrc": "2654",
      "mat_id": "ICCS 001"
    }
  ]
}

HTTP status code 400

Body

Media type: application/json

Type: any

Example:

{
  "error": "parametros de query no reconocidos"
}

Obtener Horario

endpoint principal que retorna el horario recomendado segun las especificaciones del usuario.

/horario post

post

Retorna el horario recomendado dadas las especificaciones del usuario

post /horario

Retorna el horario recomendado dadas las especificaciones del usuario

  • Request
  • Response

Headers

  • access_token: required(string)

    el token devuelto al iniciar sesion (Authorization bearer)

Body

Media type: aplication/json

Type: object

Properties
  • matricula: required(string)

    matricula del usuario

  • hra_inicio: required(string)

    cadena con formato de 24 horas. "HH:00"

    Example:

    08:00
  • hra_final: required(string)

    cadena con formato de 24 horas. "HH:00"

    Example:

    17:00
  • materias_obligadas: required(array of )

    lista de materias que son forzosamente requeridos por el usuario para armar el horario. Puede contener una lista vacia Cada materia es un objeto compuesto del mat_id y el nrc con valor opcional NA en caso de que no sea requerido.

    Example:

    [
  {
    "mat_id": "ICCS 001",
    "NRC": "971564"
  },
  {
    "mat_id": "ICCS 006",
    "NRC": "NA"
  }
]

Examples:

sinMateriasRequeridas:

{
  "matricula": "201750505",
  "hra_inicio": "08:00",
  "hra_final": "15:00",
  "materias_obligadas": []
}

conMateriasRequeridas:

{
  "matricula": "201750505",
  "hra_inicio": "08:00",
  "hra_final": "15:00",
  "materias_obligadas": [
    {
      "mat_id": "ICCS 001",
      "NRC": "971564"
    },
    {
      "mat_id": "ICCS 006",
      "NRC": "NA"
    }
  ]
}
